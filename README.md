# 📈 Stock Market Analysis & Prediction

## 🔥 Overview
This project analyzes stock market data of multiple companies and builds a machine learning model to predict stock prices.

## 🚀 Features
- Data Cleaning & Preprocessing
- Time-Series Analysis
- Moving Average (MA10, MA20)
- Buy/Sell Signal Detection
- Volatility & Risk Analysis
- Machine Learning Prediction

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## 📊 Live Demo
👉 https://your-app.streamlit.app

## 📂 Dataset
Kaggle Stock Market Dataset
Dataset available on Kaggle:
https://www.kaggle.com/datasets/jacksoncrow/stock-market-dataset

## 💡 Key Insights
- Identified high-risk and high-return stocks
- Built prediction model using regression
- Visualized trends and correlations
